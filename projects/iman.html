IMAN.

Aesthetic Coffee Shop Platform

The Project

"Cascade" is an aesthetic, anime-style coffee shop web application originally conceptualized in Figma. The goal was to translate a highly stylized visual design into a fully functional e-commerce environment, prioritizing both user experience and backend reliability.

The Deployment

Developed a working database architecture that seamlessly powers a functional "add to cart" system. This ensures that the digital storefront not only looks beautiful but operates flawlessly as a complete, data-driven web shop.
